# 🎮 Aether

> Un RPG de mundo abierto de nueva generación.

## Descripción

Aether es un videojuego RPG de mundo abierto en desarrollo.

Nuestro objetivo es crear un universo vivo con exploración, combate, ciudades, economía dinámica, inteligencia artificial avanzada y una historia profunda.

## Estado

🚧 En fase de preproducción.

## Tecnologías previstas

- Unreal Engine 5
- C++
- Blueprints
- Blender
- Git
- GitHub

## Objetivo

Crear un RPG de calidad profesional documentado desde cero.
