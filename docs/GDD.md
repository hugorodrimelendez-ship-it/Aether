# Visión del proyecto

Aether es un RPG de mundo abierto en el que el jugador explora un universo vivo, donde las decisiones tienen consecuencias y los sistemas del mundo evolucionan de forma dinámica.

---

# Objetivos

- Crear un mundo inmersivo.
- Diseñar una economía dinámica.
- Implementar IA avanzada para NPC.
- Desarrollar una historia profunda.
- Ofrecer libertad de exploración.
