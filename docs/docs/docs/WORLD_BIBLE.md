# Filosofía del Mundo

Aetheria no gira alrededor del jugador.

El jugador es una persona más en un mundo con miles de años de historia.

Las guerras continúan sin él.
Las ciudades crecen sin él.
Los reyes nacen y mueren sin él.

# Principios del Diseño

- El mundo siempre cambia.
- Ninguna ciudad existe sin historia.
- Cada criatura pertenece a un ecosistema.
- Cada reino tiene intereses propios.
- El jugador puede cambiar el mundo, pero nunca es el centro del universo.
