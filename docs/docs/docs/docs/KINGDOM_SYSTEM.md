# KINGDOM SYSTEM
## Sistema de Reinos

Versión: 0.1.0

---

# Objetivo

Permitir al jugador crear, administrar y expandir su propio reino.

---

# Evolución

Nivel 1: Campamento

- 1-10 habitantes

Nivel 2: Aldea

- 10-50 habitantes

Nivel 3: Villa

- 50-200 habitantes

Nivel 4: Ciudad

- 200-1000 habitantes

Nivel 5: Capital

- Más de 1000 habitantes

---

# Gestión

El jugador podrá:

- Construir edificios.
- Nombrar gobernadores.
- Cobrar impuestos.
- Gestionar recursos.
- Reclutar soldados.
- Establecer leyes.
- Crear rutas comerciales.

---

# Objetivo final

Construir un reino que sobreviva y prospere a lo largo de generaciones.
