# PLAYER
## Diseño del Jugador

Versión: 0.1.0

---

# Filosofía

El jugador no es un héroe predestinado.

Es una persona que llega al continente con la posibilidad de construir su propia historia.

---

# Inicio

Equipo inicial:

- Mochila
- Ropa sencilla
- Hacha de hierro
- Cuchillo
- 10 monedas
- Tienda de campaña
- Caballo

---

# Atributos

- Salud
- Resistencia
- Fuerza
- Agilidad
- Inteligencia
- Carisma

---

# Habilidades

No existen clases.

El jugador mejora usando las habilidades.

Ejemplos:

- Minería
- Espadas
- Arquería
- Cocina
- Herrería
- Comercio
- Construcción
- Agricultura
- Liderazgo

---

# Objetivo

El jugador decide su propio destino.

Puede convertirse en:

- Rey
- Mercader
- Aventurero
- Constructor
- General
- Explorador
- Artesano
