# CITIES
## Sistema de Ciudades

Versión: 0.1.0

---

# Filosofía

Las ciudades no son solo lugares para comprar objetos.

Cada ciudad tiene una historia, una economía, una cultura y una personalidad.

El jugador debe reconocer una ciudad solo por su arquitectura, su música y la forma en la que viven sus habitantes.

---

# Tipos de ciudades

## Aldea

Habitantes: 20 - 100

Edificios:

- Casas
- Granja
- Pozo
- Molino
- Taberna

---

## Villa

Habitantes: 100 - 500

Edificios:

- Mercado
- Herrería
- Iglesia o Templo
- Cuartel
- Posada

---

## Ciudad

Habitantes: 500 - 5000

Edificios:

- Murallas
- Puerto
- Palacio
- Biblioteca
- Universidad
- Gremios
- Hospital

---

## Capital

Habitantes: más de 5000

Incluye:

- Castillo Real
- Distrito Comercial
- Distrito Noble
- Barrio Artesano
- Puerto Internacional
- Academia Militar
- Catedral
- Gran Mercado
- Banco Nacional

---

# IA de la Ciudad

Cada ciudad tendrá:

- Economía propia
- Impuestos
- Producción
- Consumo
- Criminalidad
- Felicidad
- Seguridad
- Comercio exterior

---

# Evolución

Las ciudades pueden crecer, prosperar, entrar en crisis o desaparecer según las decisiones del jugador y los acontecimientos del mundo.

---

# Objetivo

Que cada ciudad se sienta como un lugar único y vivo, con identidad propia.
