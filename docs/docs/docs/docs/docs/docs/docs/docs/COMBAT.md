# COMBAT SYSTEM
## Proyecto Aether

Versión: 0.1.0

---

# Filosofía

El combate debe ser táctico, desafiante y basado en la habilidad del jugador.

No habrá enemigos con millones de puntos de vida.

Una buena estrategia será más importante que el nivel.

---

- Espada y escudo
- Hacha
- Lanza
- Arco

- Magia
- Combate sin armas

# Estadísticas

- Salud

# Estados

Los personajes pueden sufrir:

- Sangrado
- Veneno
- Congelación
- Quemaduras
- Aturdimiento
- Miedo
- Cansancio

# Inteligencia Artificial

Los enemigos podrán:

- Esquivar ataques
- Bloquear
- Rodear al jugador
- Pedir ayuda
- Huir si tienen miedo
- Proteger a sus aliados

---

# Tipos de enemigos

- Animales
- Bandidos
- Soldados
- Bestias
- Criaturas legendarias
- Jefes

---

# Objetivo

Que cada combate sea diferente y obligue al jugador a pensar antes de actuar.
