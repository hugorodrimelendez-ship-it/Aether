# RACES
## Razas del Mundo de Aether

Versión: 0.1.0

---

# Filosofía

Las razas no existen solo para cambiar la apariencia del personaje.

Cada raza tiene:

- Historia propia.
- Cultura.
- Arquitectura.
- Idioma.
- Tradiciones.
- Ventajas.
- Desventajas.
- Relaciones con otras razas.
